Durante los ultimos años, el estandar HTML5 se ha estado aafianzando en muchas paginas web. Sin embargo, ¿que podemos decir con eso? ¿Cual es la diferencia crucial entre HTML "común", por asi decrilo, y este nuevo estandar? Hoy hablaremos sobre esto y responderemos a la pregunta de qué es HTML5

HTML 5 es un lenguaje mas semántico. En otras palabras, es un lenguaje que permite a los desarrolladores web describir con mayor precisión el contenido de una pagina web., se dota de un significado a las diferentes partes de la pagina Wb, de forma que el navegador puede distinguir la parte que forma parte de la cabecera gracias a que hay una etiqueta que dice "cabecera", bueno lo dice en ingles "header". ¿PAra que sirve? Ayuda a los buscadores como Google a entender la pagina Web, algo que se espera sea positivo para el SEO. y tambien para que los navegadores muestren de forma inteligente diferentes versiones de la pagina Web sin tener que usar plugins para dispositivos especificos.

Introduccion a CSS3

El nombre hojas de estilo en cascada viene del ingles cascadign style sheets, del que toma sus siglas. CSS es un lenguaje para definir la presentacion de un documento estructurado escrito en HTML. CSS es el lenguaje que describe como se debe mostrar un documento escrito en HTML.

¿Para qué sivre el CSS3?

El CSS sirve para definir la estética de un sitio web en un documento externo y eso mismo permite que modificando este documento se pueda cambiar la estetica entera de un sitio web. El sito puede variar totalmente de estetica cambiando unicamente solo el CSS, sin toca para nada los documentos HTML que lo componen.
